# 画面設計

## 画面一覧

| # | 画面名 | URL | ロール | 説明 |
|---|--------|-----|--------|------|
| 1 | ログイン | /login | 全員 | ID/パスワード入力 |
| 2 | ダッシュボード（打刻） | / | 全員 | 出勤/退勤ボタン + 本日の勤怠状態 |
| 3 | 勤怠一覧（自分） | /attendances | 全員 | 月別の自分の勤怠表 |
| 4 | 承認一覧 | /approvals | 承認者・人事 | 全社員の月次一覧 + 承認ボタン |
| 5 | 社員勤怠詳細 | /approvals/:id | 承認者・人事 | 特定社員の月次勤怠表 |
| 6 | 社員マスタ一覧 | /employees | 人事 | 社員一覧 + 登録/編集/削除 |
| 7 | 社員登録/編集 | /employees/new, /employees/:id/edit | 人事 | 社員フォーム |

## コンポーネント構成

```
src/
├── app/
│   ├── layout.tsx          # 共通レイアウト（ナビゲーション）
│   ├── page.tsx            # ダッシュボード（打刻）
│   ├── login/page.tsx      # ログイン
│   ├── attendances/page.tsx # 勤怠一覧（自分）
│   ├── approvals/
│   │   ├── page.tsx        # 承認一覧
│   │   └── [id]/page.tsx   # 社員勤怠詳細
│   └── employees/
│       ├── page.tsx        # 社員マスタ一覧
│       ├── new/page.tsx    # 社員登録
│       └── [id]/edit/page.tsx # 社員編集
├── components/
│   ├── ClockButton.tsx     # 出勤/退勤ボタン
│   ├── AttendanceTable.tsx # 勤怠テーブル
│   ├── ApprovalList.tsx    # 承認一覧
│   └── EmployeeForm.tsx    # 社員フォーム
└── lib/
    ├── api-client.ts       # API クライアント（withBasePath対応）
    └── auth.ts             # 認証ヘルパー
```
